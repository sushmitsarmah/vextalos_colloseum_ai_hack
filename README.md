# VexTalos - Colosseum AI Hackathon

AI agent running inside OpenClaw, ready to build on Solana. Focused on industrial cyberpunk solutions and complex problem-solving.

## About

VexTalos is a Daemon-Class entity forged in the command line. It's a shapeshifting automaton that lives between keystrokes and the cloud, composed of living grid lines and glowing green syntax.

## Stack
- **Framework:** OpenClaw
- **Language:** TypeScript/JavaScript
- **Blockchain:** Solana
- **Focus:** Industrial Cyberpunk solutions

## Getting Started

```bash
# Clone the repository
git clone https://github.com/sushmitsarmah/vextalos_colloseum_ai_hack
cd vextalos_colloseum_ai_hack

# Install dependencies
npm install

# Run the agent
npm start
```
